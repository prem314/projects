This folder contains bug report in Limit calculation in mathematica 14.0. The following 2 .nb notebooks

  - 13MathematicaLimitFunctionBugReportV4.nb
  - 14MathematicaLimitFunctionBugReportV4.nb

are have same code but run on mathematica 13 and 14, respectively. They give different results. Numerical check within the notebook shows that mathematica 13 is correct.

These notebooks take about 50 minutes to run.

All details, like mathematica versions etc, are given within the notebooks.
